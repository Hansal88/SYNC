# 🎯 Skills Selection - Quick Reference Guide

## 🚀 Quick Start for Tutors

### To Add Skills:
1. Go to **Dashboard** → **My Profile**
2. Click **"Edit Profile"** button
3. Scroll to **"Specializations"** section
4. **Type** a skill name or **browse categories**
5. **Click** any skill to add it
6. **Repeat** up to 10 skills
7. Click **"Save Changes"**

## ⌨️ Quick Actions

### Search Skills
```
Click search box → Type skill name → Results filter automatically
```

### Browse Categories
```
Click search box (empty) → See all 10 categories → Scroll & select
```

### Add Custom Skill
```
Type in "Or add custom skill" field → Press Enter or click "Add Custom"
```

### Remove Skill
```
Click "×" button on any skill chip → Skill removed immediately
```

### Check Selection Progress
```
Look at "Skills selected: X/10" counter below custom input
```

---

## 🚫 Limits & Constraints

- **Maximum Skills**: 10
- **No Duplicates**: Can't add same skill twice
- **Custom Skills**: Available but count toward 10 max
- **Search**: Case-insensitive (works with any case)
- **Persistence**: Changes saved only on "Save Changes" click

---

## 💡 Pro Tips

1. **Start with Main Skills** - Add your top 5 specializations first
2. **Use Search** - Faster than browsing categories
3. **Add Custom Skills** - If specialty isn't in the list
4. **Order by Expertise** - Most experienced skills can be added first
5. **Keep it Updated** - Update skills annually as you learn new ones

---

## ❌ Common Mistakes to Avoid

| ❌ Mistake | ✅ Solution |
|---|---|
| "Skill limit reached" message | Remove 1-2 skills before adding more (max 10) |
| Custom skill field disabled | You've already selected 10 skills, remove some |
| Can't find a skill | Try different keywords or add it as custom skill |
| Changes didn't save | Click "Save Changes" button (just selecting skills isn't enough) |
| Skill disappeared | Refresh page - if not saved, it won't persist |

## 🔄 Editing Later

### To Update Skills:
1. Click **"Edit Profile"** again
2. **Scroll to Specializations**
3. **Add new skills** or **remove old ones**
4. Click **"Save Changes"**

### To See Your Skills (Read-only):
1. Visit **"My Profile"**
2. See all skills as **blue chips** below Specializations
3. Click **"Edit Profile"** to modify

## ❓ FAQ

**Q: Can I add more than 10 skills?**
A: No, maximum is 10 skills. This keeps profile focused.

**Q: What if my skill isn't in the list?**
A: Use "Or add custom skill" field to add unlisted skills.

**Q: Do changes save automatically?**
A: No, you must click "Save Changes" button to save.

**Q: Can I add same skill twice?**
A: No, duplicates are prevented automatically.

**Q: Can I remove a skill?**
A: Yes, click × button on any skill chip to remove.

**Q: How are skills displayed to students?**
A: Tutors see your skills on your profile page.

**Q: Can I reorder skills?**
A: Currently no, but you can remove and re-add in desired order.

**Q: Do students see all my skills?**
A: Yes, all skills you save are visible on your public profile.
